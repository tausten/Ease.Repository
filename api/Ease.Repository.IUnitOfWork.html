CompleteAsync()

Wrap up the unit of work (i.e. success path). If a IUnitOfWork is Disposed before it has been Completed, then the work is considered abandoned and should be undone. After the unit of work has been Completed, it is invalid to continue with more work prior to Dispose, and Complete may perform an implicit Dispose to help enforce this.
